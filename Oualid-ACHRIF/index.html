Meta

Meta is HTML tag used to provide additional information about a page to search engines and other clients.

charset="UTF-8"

Charset="UTF-8" is used to specify the character encoding of an HTML document. It ensures that the webpage can accurately represent various characters from different languages.

viewport :

name="viewport" is like a set of instructions for web browsers. It helps make sure that the web page looks good and is easy to use on different screen sizes (mobile, tablet, laptop, desktop).

content="width=device-width, initial-scale=1.0" :

- width=device-width helps the webpage to adjust its size based on the screen width of the device.
- initial-scale=1.0 means that the webpage is displayed at its normal size, It ensures that the initial view of the webpage is not zoomed in or out,
